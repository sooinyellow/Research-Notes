# Configuration Space (C-Space)

## Definition

The configuration space is the set of all possible configurations of a robot.

## Key Concepts

- Configuration
- Degrees of Freedom (DOF)
- Dimension

## Example

A point robot moving on a plane has a 2D configuration space.

## My Understanding

C-space represents all possible states of a robot.
The dimension of the C-space is equal to the number of DOF.
